# **Deep Splicing**
This code is used to apply Deep Learning framework for image foresnics (splicing detection)


----------

- Python 3.4.4
- [Theano](http://deeplearning.net/software/theano/)

- [pwprutils](https://github.com/paolorota/pyprutils)
